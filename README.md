# VibeJournal 🌀

A topological documentation layer for vibe-coded projects. Captures the "why" behind every code change.

## Installation

```bash
npm install https://github.com/onedownz01/vibe-journal
```

## Usage

### Initialize
```bash
vibe-journal init
```

### Log Intent
```bash
vibe-journal log "Intent" "Location" "Implementation" "Trigger" "Classification"
```

## Taxonomy
- **UI Polish**: Visual adjustments.
- **Logic Refactor**: Code structure changes.
- **Data Structure**: Changes to models or schemas.
- **Dependency Injection**: Adding tools or integrations.
- **Debug Patch**: Bug fixes.
- **Experiment**: Temporary explorations.
